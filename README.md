## :point_right: 프로젝트 소개 및 진행 범위

홈페이지내 메인페이지를 대상으로 하였으며,<br>
프로젝트 의도에 따라 자바스크립트가 필요한 기능은 제외하였습니다.<br>
** 실제 사이트의 경우 새로고침시 텍스트가 변동되는 영역이 있습니다. 구현 사이트는 작업 당시의 내용을 기준으로 하였습니다.
